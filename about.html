Tentang AnimGallery

AnimGallery Pro adalah proyek eksperimen CSS dan JS vanilla yang ditujukan untuk belajar cara membuat animasi web, transisi, serta manajemen state sederhana menggunakan localStorage (seperti Dark Mode).

Dalam versi ini, kode telah di-refactor menjadi arsitektur "multi-halaman" konvensional. Kami telah memisahkan berkas HTML, CSS, dan JavaScript secara independen, sehingga kodenya menjadi lebih rapi dan lebih mudah untuk dikelola saat aplikasi terus berkembang.

Terus berkreasi dan selamat belajar! 🚀

Built with ❤️ using HTML, CSS, and JS (Multi-page Edition).
